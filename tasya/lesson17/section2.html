Walkthrough Soal Bareng

Kita selesaikan 2 contoh soal lengkap — dari nulis sistemnya, gambar garis, arsir daerah, sampai nemuin irisannya dan verifikasi pakai uji titik.

📌 Contoh Soal 1

Tentukan daerah solusi sistem berikut:

y > x − 1
y < 2x + 2

Kita selesaikan langkah demi langkah. Ingat prosesnya: gambar garis → tentukan arsiran → cari irisan → uji titik.

Langkah 1 & 2 — Gambar kedua garis

Ubah ke persamaan garis dulu

1
Pertidaksamaan 1: y > x − 1
→ gambar garis y = x − 1 (putus-putus karena > bukan ≥)
2
Pertidaksamaan 2: y < 2x + 2
→ gambar garis y = 2x + 2 (putus-putus karena < bukan ≤)
3
Titik potong kedua garis: x − 1 = 2x + 2 → x = −3, y = −4
Titik potong: (−3, −4)
Langkah 3 — Tentukan Arsiran

Uji titik asal (0, 0) untuk tiap pertidaksamaan

PertidaksamaanUji (0,0)HasilArsiran
y > x − 10 > 0 − 1 → 0 > −1✅ BenarDi atas garis y = x−1
y < 2x + 20 < 2(0) + 2 → 0 < 2✅ BenarDi bawah garis y = 2x+2
Titik (0,0) ada di kedua daerah arsiran → ini artinya titik asal termasuk ke dalam daerah irisan!
Langkah 4 — Daerah Solusi

Irisan: daerah yang memenuhi keduanya

Daerah solusi = area hijau = semua titik yang berada di atas y = x−1 sekaligus di bawah y = 2x+2.

Verifikasi ulang dengan titik (0,0): ✅ 0 > −1 dan ✅ 0 < 2 → memenuhi keduanya!
📌 Contoh Soal 2 — Lebih Menantang

Tentukan daerah solusi sistem:

y ≥ −x + 1
y ≤ x + 3
x ≥ 0

Kali ini ada 3 pertidaksamaan! Prosesnya sama, tapi kita arsir 3 daerah dan cari irisannya.

Coba kamu pikirkan dulu: titik (1, 2) — apakah memenuhi ketiga syarat?

SyaratCek titik (1,2)Status
y ≥ −x + 12 ≥ −1+1 = 0✅
y ≤ x + 32 ≤ 1+3 = 4✅
x ≥ 01 ≥ 0✅
Titik (1,2) memenuhi semua syarat → ada di daerah solusi! ✅
Visualisasi Soal 2

3 pertidaksamaan → daerah segitiga

1
y ≥ −x + 1 → di atas garis y = −x+1
2
y ≤ x + 3 → di bawah garis y = x+3
3
x ≥ 0 → di kanan sumbu y (x positif)
Irisan ketiga daerah ini membentuk sebuah segitiga. Semakin banyak syarat, semakin "sempit" daerah solusinya!
📋 Rangkuman Langkah

Cara menyelesaikan sistem pertidaksamaan

1
Tulis setiap pertidaksamaan sebagai persamaan garis
2
Gambar semua garis (putus-putus jika < atau >, solid jika ≤ atau ≥)
3
Uji titik (biasanya titik asal) ke tiap pertidaksamaan → tentukan sisi mana yang diarsir
4
Temukan area yang arsiran semua pertidaksamaannya overlap → itu daerah solusinya
5
Verifikasi satu titik di area irisan ke semua pertidaksamaan
